# Payment Track App
A simple mobile app to track monthly payments with a spreadsheet-like interface

### Features
-Login with username/password
-Spreadsheet-like layout with the following column:
    -Budget Name
    -Amount
    -last Payment Date
    -App Use
    -Paid(Yes/No)
    -Paid Date
-Date stored using SQLite
